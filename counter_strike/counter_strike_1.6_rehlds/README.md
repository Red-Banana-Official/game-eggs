# Counter-Strike 1.6 - ReHLDS

Description :
CS 1.6 ReHLDS binaries egg.

ReHLDS is a reverse-engineered, optimized version of HLDS.
Developed by third-party modders to improve stability, performance, and security.
Fully compatible with MetaMod, AMX Mod X, and other plugins.

Included modules are rehlds, amxmodx, metamod-r, reapi, ReGameDLL_CS, reunion and revoice-plus
